#  Smart Lighting System Based on Light Intensity  

##  Overview  
An automated lighting system that controls lights based on ambient light intensity using sensors and a microcontroller. It helps reduce energy consumption by turning lights on only when needed.  

---  

##  Features  
-  Automatic light control using LDR sensor  
-  Energy-efficient operation  
-  Adjusts lighting based on surroundings  
-  LED indication/output control  
-  Simple and cost-effective design  

---  

##  Components Used  
- Arduino / Microcontroller  
- LDR (Light Dependent Resistor)  
- Resistors  
- LED / Light Source  
- PIR Sensor 
- Power Supply  

---  

##  Working  
The LDR sensor detects the ambient light intensity. When the surrounding light level falls below a certain threshold (dark condition), the microcontroller turns ON the light. When sufficient light is detected (bright condition), it automatically turns OFF the light, ensuring efficient energy usage.
